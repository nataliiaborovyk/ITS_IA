# Lezioni IA.1

Questo progetto contiene il codice Python delle lezioni dell'unit√† IA.1

Una volta clonato il progetto sulla propria macchina avente l'ecosistema Docker dell'ITS (disponibile su https://github.com/ITS-ICT-Academy/sw_development/tree/2024-2026) propriamente installato e attivo, eseguire i passi di seguito:

1) docker exec -it its_dev bash

3) (all'interno del container its_dev) cd test/codice
4) (per verificare il funzionamento del pacchetto pandas) python pandas_test.py
5) (per verificare il funzionamento del pacchetto matplotlib) python matplotlib_test.py
